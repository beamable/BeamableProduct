# Package Installer

This directory contains the legacy Unity-based Package Installer used to create the Beamable SDK `.unitypackage` installer. Most users should install Beamable via the scoped UPM registry; this folder is for maintainers who need to rebuild or modify the installer.

If you only need to consume the SDK, follow the instructions in `client/Packages/README.md` or the main repository docs.
