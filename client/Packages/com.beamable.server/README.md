# Beamable Server (deprecated)

This package is deprecated and no longer contains runtime code. Server-side types and functionality have been consolidated into the `com.beamable` package and the shared `Beamable.Common` project.

If you are looking for the shared .NET types, see:

`client/Packages/com.beamable/Common` — the canonical Common project (published as `Beamable.Common` on NuGet).

For CLI-specific behavior see `cli/cli`.
