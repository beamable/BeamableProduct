# iOS Native Libraries

Planned — not implemented yet.

This folder will hold the iOS (Swift / Objective-C) equivalents of the Android libraries:
- **PushNotifications** — local + remote (APNs) push.
- **Deeplink** — Universal Links / custom URL scheme capture.

For now, the Unity project keeps using its existing iOS implementation
(`com.unity.mobile.notifications` + `Application.deepLinkActivated`).
